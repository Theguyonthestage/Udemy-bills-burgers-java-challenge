# Bills Burgers Java Challenge

Object-oriented Java project implementing a burger ordering system.

## Features
- Base Hamburger with up to 4 additions
- HealthyBurger with 2 extra healthy additions
- DeluxeBurger with automatic chips and drink (no extra additions allowed)
- Receipt printing with total price calculation

## Concepts Used
- Inheritance
- Method overriding
- Encapsulation
- Basic OOP design
